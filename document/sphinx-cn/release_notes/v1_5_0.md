# v1.5.0

**⭐️ 重要功能**

**✨ 次要功能和优化**

- ros2 插件
  - 优化启动流程，将回调处理移至 start 阶段，确保正常接收 transaction_local 消息
  - 新增 events_executor 执行器
- 录播插件：
  - 修改 signal 模式下的包创建逻辑
  - 新增接口：获取录制 action 的状态
- rotate_file 日志增加 suffix_with_timestamp 选项以在日志文件名后缀增加该日志的旋转时间
- 日志新增 enable_crash_log 配置，可控制程序崩溃时是否输出堆栈信息

**🐛Bug 修复**

- 修复：录播插件对于 AimRT v1.3 之前版本的兼容性问题
- 修复：修正因时序导致的 ros2 后端的 rpc 超时
